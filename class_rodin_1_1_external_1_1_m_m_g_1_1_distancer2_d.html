Computes a signed distance function to a subdomain.

Parameters

Returns Signed distance function representing .

This function generates the signed distance function to a domain which is contained in the given bounding box , where is identified by its material reference. By default, the elements of are assumed to be specified by the number 3.

ScalarSolution2D Rodin::External::MMG::Distancer2D::distance(Mesh2D& box, Mesh2D& contour) const

Computes a signed distance function from a given bounding box and contour.

Parameters
box in Bounding box containing the contour.
contour in Orientable contour to distance.
Returns Signed distance function representing .

This function generates the signed distance function to a domain supplied by means of a mesh of its boundary , at the vertices of a computational mesh of a bounding box .

Note

The contour mesh is allowed to contain a volume part, in which case only the edge (2D) or triangle (3D) information will be retained.

Given a level set function defined on the vertices of a bounding box , this method will regenerate the signed distance function associated to the the subdomain which the level set function represents.

Distancer2D& Rodin::External::MMG::Distancer2D::setActiveBorders(const std::set<MaterialReference>& refs)

Specifies the active borders of the mesh.

Returns Reference to self (for method chaining)

An active border (in 2D) is a subset of the bounding box boundary where the level set function will be equal to zero.

This implicitly enables active border mode.

By default, all the bounding box boundary is inactive.
