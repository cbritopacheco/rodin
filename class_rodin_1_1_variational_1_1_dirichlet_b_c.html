Rodin::Variational::DirichletBC class

Represents a Dirichlet boundary condition.

Contents

  • Example
  • Reference
    • Constructors, destructors, conversion operators

Imposing a Dirichlet boundary condition amounts to projecting the boundary value over the function on the true boundary degrees of freedom. In simpler terms, the following condition will be satisfied on the segment of the whole boundary specified.

Note

This class is meant to be instanced as a part of a Variational::FormLanguage expression.

Example

Take the simple Poisson equation with Dirichlet boundary conditions:

Integrating by parts we arrive at the variational formulation.

In code this will look like:

Problem poisson(u, v);
poisson = DiffusionIntegrator(lambda)
        - DomainLFIntegrator(1.0)
        + DirichletBC(GammaD, 0.0);
